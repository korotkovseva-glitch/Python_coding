It's a assignment work for cooding for economist class.
There was choosed the hotels vienna data set to demonstrate a learning outcomes.
To access the data set please use the following link: https://osf.io/4e6d8/overview
The learning outcomes cosider a step by step solution for each point.
Folder raw data includes a csv file of accomodations in Vienna in raw format from OSF.
Folder Code_py includes 2 codes: 1. Filteting of data, 2. Analysis of data: summary statistics and graphs.
The version of raw data after cleaning in phyton is located in the filder Clean_data. The graph outputs from phyton are included in the folder Graphs_Py.
Then the filtered version of csv has been used in Stata. There is a folder of do files in stata with master file. Stata coding has included working with data, analysis (statistics), graphs. After additional corrections of csv file, it was saved in Stata's dta format. It's located in Clean_data folder.
The folder Outputs_stata includes the summary statistic tables after data analysis and graphs in Stata
